Education — Software Development Services

Educational software is software intended for use in teaching and self-learning environments. Intelli Infotech specialises in rich UI applications and is therefore able to turn complex educational materials into effective educational software. Any knowledge — from textbooks to test materials — can be presented in the form of software and distributed online, making education more and more accessible to everyone.

Courseware

- Educational materials intended for use on a computer, including lessons, tests, and other materials that require online availability

Classroom Aids

- Educational materials intended for projection onto large boards or running simultaneously on a network of desktop computers in a classroom
- Bright, flash-animated educational materials designed for group learning environments

Corporate Training & University / Post-Graduate Education Software

- Online educational services and systems that allow necessary educational materials to be distributed, results to be collected, and progress to be displayed to teaching staff

Educational Web Portals

- Online educational services for every level of education
- Textbooks, educational materials, and case study readers
- Test generation and online test taking
- Communication systems with teachers and social networking integration

Mobile Educational Software

- Mobile versions of educational portals optimised for iPhone, Windows Mobile, Windows Phone, and BlackBerry devices
